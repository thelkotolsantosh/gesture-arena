# Security Policy 🛡️

We take the security of Gesture Arena seriously. If you find a security issue or vulnerability, please let us know immediately.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please do **NOT** report it publicly through a GitHub Issue. Instead, follow these steps:

1. Send an email describing the vulnerability to the repository maintainer (or use the GitHub Private Vulnerability Reporting feature if enabled).
2. Include the following details to help us investigate:
   * A description of the issue.
   * Steps to reproduce the issue.
   * Any potential impact or risk.

We will try to review your report within 48 hours and work on a fix as quickly as possible. Once the fix is released, we will credit your contribution.
