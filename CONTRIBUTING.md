# Contributing to Gesture Arena 🤝

Thank you for wanting to contribute to Gesture Arena! We welcome all contributions to make this game even better. 

Here are some simple guidelines on how you can help:

---

## 1. Reporting Bugs 🐛
If you find a bug in the game (like a gesture not tracking or screen freezing):
1. Check if the issue is already reported in the repository.
2. If not, open a new **Issue**.
3. Describe the bug simply:
   * What happened?
   * What did you expect to happen?
   * What browser and webcam are you using?

## 2. Suggesting Features 💡
Have a cool idea for a new hand gesture or ultimate ability?
1. Open a new **Issue**.
2. Explain your idea:
   * What gesture should trigger it?
   * What does the ability do on screen?
   * Why would it be fun?

## 3. Pull Requests (Code Contributions) 🛠️
If you want to write code to fix a bug or add a feature yourself:
1. Fork the repository to your own account.
2. Create a new branch for your changes:
   ```bash
   git checkout -b my-cool-feature
   ```
3. Make your changes and test them locally using `npm run dev`.
4. Run `npm run build` to verify there are no TypeScript or compilation errors.
5. Push your branch and open a **Pull Request** to our main branch.
6. Provide a simple description of what your code does in the pull request.

---

## Code Style Guidelines
* Write clean, readable code.
* Use TypeScript types properly (avoid using `any` where possible).
* Keep your commits small and write simple, descriptive commit messages.
